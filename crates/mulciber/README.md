# Mulciber

Mulciber is a research-stage native game platform for modern Vulkan and Metal systems. The project
is validating native resource, rendering, presentation, and lifecycle implementations before it
extracts a stable public graphics API.

Version 0.1.0 intentionally contains only the documented library shell. It is published so the
canonical package points to the active project while backend contracts are still being established;
it is not ready for application use.

Development and runnable probes live in the
[Mulciber repository](https://github.com/fairhill1/mulciber).
